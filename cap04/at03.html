Seção 3 — Glossário Técnico

HTML
Linguagem de marcação para páginas web.
CSS
Linguagem usada para estilos.
JavaScript
Linguagem de programação da web.
Também pode ser usada no backend.
API
Interface de Programação
Permite comunicação entre sistemas.
HTTP
Protocolo usado na internet.
URL
Endereço Web
Localização de um site ou página.
DNS
Converte domínio em IP.
